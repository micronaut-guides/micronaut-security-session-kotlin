== Create an App with the Command Line Interface

Create an app using the http://docs.micronaut.io/snapshot/guide/index.html#cli[Micronaut Command Line Interface].

`mn create-app example.micronaut.complete --lang=kotlin --test=spock`

The previous command creates a Micronaut app with the default package `example.micronaut` in a folder named `complete` which uses Kotlin as its main programming language and spock as a Test Framework.
